📊 一、开仓条件优化

路径1：主策略（精简为6个核心条件）

核心条件（全部满足）：

  1. 无持仓
  2. 不在冷却期（冷却期缩短为10分钟）
  3. 5分钟信号不为HOLD
  4. trend1h方向 = 信号方向（EMA差>0.3%，提高阈值）
  5. 信号评分 ≥ 50分（原45分）
  6. AI置信度 ≥ 60（原58）
移除的条件（容易过度过滤）：
  • 移除"不在K线收盘前30秒"（频繁波动是正常的）
  • 移除"同方向连续信号<3次"（趋势可能持续）
  • 移除"每日盈亏限"（用仓位管理替代）

路径2：大单信号（增加确认机制）

  1. 订单簿买方占比从<45%突变到>75%（做多）
    或卖方占比从<45%突变到>75%（做空）
  2. 成交量>上次1.5倍（原2倍，降低阈值）
  3. 1h趋势与方向一致
  4. + 新增：突变后等待1根5分钟K线确认 避免假突破
  5. 不在重要数据窗口
  6. 未开启AI主导模式

路径3：AI主导（增加人工确认）

改为"AI建议 + 人工确认"模式：

  1. 勾选"AI主导模式"
  2. AI置信度 ≥ 65（原58，提高阈值）
  3. + 新增：信号连续2次一致方向（原1次） 减少噪音
  4. 人工确认开关（可选）

💰 二、止损/止盈优化（核心改动）

新盈亏比目标：至少 3:1

EMA差 止损 止盈 盈亏比
> 0.6% 2% 7% 3.5:1
> 0.3% 1.5% 5% 3.3:1
< 0.3% 1% 3.5% 3.5:1
改动点： 止盈全面提高，不再追求"高胜率低盈亏"，改为"低胜率高盈亏"思维

大单策略（路径2）

项目 原值 新值
止损 0.6% 0.8%
止盈 1.5% 2.4%
盈亏比 2.5:1 3:1

AI置信度 ≥ 65（原58） AI_CONFIRM_COUNT = 2 # 连续2次信号确认（原1次） # 止损止盈（盈亏比3:1+） # 模式: (止损%, 止盈%, 最小EMA差) SL_TP_RULES = [ (0.02, 0.07, 0.006), # EMA差>0.6% → 止损2% 止盈7% (0.015, 0.05, 0.003), # EMA差>0.3% → 止损1.5% 止盈5% (0.01, 0.035, 0.0), # ==================== 数据结构 ==================== class TradeDirection(Enum): LONG = "long" SHORT = "short" HOLD = "hold" class SignalType(Enum): MAIN = "main" # 主策略 LARGE_ORDER = "large" # 大单信号 AI_LEAD = "ai_lead" # ==================== 核心策略类 ==================== class TradingStrategy: def __init__(self, config: Config = Config()): self.config = config self.state = StrategyState() def get_sl_tp(self, ema_diff: float): """根据EMA差获取止损止盈""" for sl, tp, min_ema in self.config.SL_TP_RULES: if ema_diff >= min_ema: return sl, tp return self.config.SL_TP_RULES[-1][:2] def get_cooldown(self, loss_pct: float) -> int: """根据亏损幅度计算冷却时间""" if loss_pct < 0.01: return self.config.COOLDOWN_SHORT elif loss_pct < 0.02: return self.config.COOLDOWN_MEDIUM elif loss_pct < 0.03: return self.config.COOLDOWN_LARGE else: return self.config.COOLDOWN_EXTREME def is_in_cooldown(self) -> bool: """检查是否在冷却期""" return time.time() < self.state.cooldown_until def calculate_position_size(self, signal_type: SignalType) -> float: """计算开仓金额""" base = self.config.BASE_POSITION_RATIO if signal_type == SignalType.LARGE_ORDER: base *= self.config.LARGE_ORDER_RATIO return base * self.config.LEVERAGE def check_protection(self): """综合保护检查""" if self.is_in_cooldown(): return False, "冷却期中" if self.state.consecutive_losses >= self.config.CONSECUTIVE_LOSS_LIMIT: return False, f"连亏{self.state.consecutive_losses}次熔断" if self.state.daily_profit >= self.config.DAILY_PROFIT_STOP: return False, "日盈利达标停止" if self.state.has_position: return False, "已有持仓" return True, "通过" def check_main_strategy(self, signal: Signal, trend_1h: TradeDirection): """主策略开仓检查""" allowed, reason = self.check_protection() if not allowed: return False, reason if signal.direction == TradeDirection.HOLD: return False, "5分钟信号HOLD" if signal.direction != trend_1h: return False, "趋势方向不一致" if abs(signal.ema_diff) < self.config.MIN_EMA_DIFF: return False, f"EMA差不足 {signal.ema_diff:.3f}" if signal.score < self.config.MIN_SIGNAL_SCORE: return False, f"信号评分不足 {signal.score}" if signal.ai_confidence < self.config.MIN_AI_CONFIDENCE: return False, f"AI置信度不足 {signal.ai_confidence}" return True, "主策略信号OK" # 检查开仓 allowed, reason = strategy.check_main_strategy( signal=sample_signal, trend_1h=TradeDirection.LONG ) print(f"开仓检查: {allowed} - {reason}") if allowed: result = strategy.open_position(sample_signal) print(f"开仓成功: {result}") if __name__ == "__main__": main()
