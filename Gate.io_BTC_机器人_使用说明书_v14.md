## 软件简介

**Gate.io BTC 量化交易机器人 v14** 是一款基于 Electron 开发的桌面端加密货币量化交易软件，专为 Gate.io 交易所的 BTC_USDT 永续合约设计。

### 主要特性
- ✅ **三模趋势过滤**：保守/激进/自适应三种模式
- ✅ **智能止盈止损**：基于 ATR 的动态计算
- ✅ **分批获利**：自动分阶段平仓锁定利润
- ✅ **移动止损**：盈利后自动跟踪保护
- ✅ **风控熔断**：自动止损保护本金
- ✅ **实时图表**：支持缩放平移的 K 线图表
- ✅ **模拟交易**：支持实盘前的策略回测验证

## 安装与启动

### 系统要求
- Windows 10/11
- Node.js 16+（开发环境）
- 网络连接（需连接 Gate.io API）

### 启动方式
1. 进入软件目录：`C:\Users\Administrator\Desktop\gate_bot_gui\`
2. 运行启动命令：`npm start`
3. 等待主窗口打开

## 交易策略详解

### 核心理念
**"大趋势定方向，小周期找入场，多重过滤防假信号"**

### 三层过滤系统

#### 第一层：趋势判断（1小时或30分钟）
- **EMA9 与 EMA21 的关系**决定趋势方向
- EMA9 > EMA21 = 做多趋势
- EMA9 < EMA21 = 做空趋势

#### 第二层：15分钟信号生成
**做多信号（满足任一条件）：**
1. 布林带下轨反弹：价格触及下轨后反弹
2. EMA金叉：EMA9上穿EMA21
3. EMA55支撑：价格在EMA55附近获得支撑

**做空信号（满足任一条件）：**
1. 布林带上轨回落：价格触及上轨后回落
2. EMA死叉：EMA9下穿EMA21
3. EMA55阻力：价格在EMA55附近遇阻

**RSI过滤**：
- 做多：RSI 30-70（避免超买区）
- 做空：RSI 30-72（避免超卖区）

#### 第三层：多时间框架确认
- 15分钟信号必须与趋势方向一致
- 顺势交易，避免逆势操作

### 信号评分系统（0-100分）
基础分50，根据以下条件加分：
- RSI超卖/超买：+15
- EMA偏离大（趋势强）：+15
- ATR高（波动大）：+10
- 看涨/看跌K线形态：+10

**开仓条件**：评分 ≥ 65 + 趋势确认通过 + 成交量确认

### 动态止盈止损（ATR-based）
```
止损 = 入场价 ± ATR值 × ATR倍数
止盈 = 入场价 ± ATR值 × ATR倍数 × 2
```
- 波动大时：止损止盈自动放宽
- 波动小时：止损止盈自动收紧

### 分批止盈逻辑
1. **第一止盈位**（默认3%）：平仓 30%
2. **第二止盈位**（默认6%）：再平仓 30%
3. **剩余40%**：使用移动止损跟踪趋势

### 保本机制
当盈利达到止损幅度时（如止损1.5%，盈利≥1.5%），自动将止损移至成本价，确保这笔交易不亏钱。

### 移动止损
盈利后自动跟随价格移动：
- 多单：价格回落 X% 时止损平仓
- 空单：价格反弹 X% 时止损平仓

## 操作指南

### 首次使用流程

#### 1. 配置 API（真实交易需要）
1. 登录 Gate.io 官网
2. 进入"API管理"创建新 API
3. 权限选择：合约交易、账户读取
4. 复制 API Key 和 Secret 填入软件
5. 勾选"记住 API"

#### 2. 设置交易参数
1. 选择交易模式（建议新手用"1h保守"）
2. 调整资金、止盈止损、杠杆
3. 查看"数学预期"确认风险收益比
4. 点击"💾 保存配置"

#### 3. 测试运行
1. **务必先开启"模拟模式"**
2. 点击"▶ 启动机器人"
3. 观察几轮信号和虚拟交易
4. 确认策略符合预期

#### 4. 实盘交易
1. 关闭"模拟模式"
2. 确认 API 已正确配置
3. 检查账户有足够 USDT（建议 ≥100U）
4. 点击"▶ 启动机器人"

### 日常操作流程
1. 启动软件，确认 API 连接状态（绿色）
2. 检查参数设置
3. 启动机器人
4. 监控运行日志和持仓状态
5. 必要时手动干预（平仓/调整止损止盈）

### 手动干预操作

#### 紧急平仓
点击持仓卡片中的"⚡ 全平"按钮，立即市价平仓。

#### 部分获利
点击"平 50%"等按钮，保留部分仓位继续博弈。

#### 调整止损止盈
在输入框填入新价格，点击确认。用于：
- 锁定更多利润
- 给波动更大空间
- 避免被假突破扫损

## 风险管理

### 内置风控机制

| 机制 | 触发条件 | 动作 |
|------|---------|------|
| 风控熔断 | 账户亏损 > 设定比例（默认15%） | 自动停止机器人 |
| 冷却期 | 止损平仓后 | 30分钟内不开新仓 |
| 保本止损 | 盈利 ≥ 止损幅度 | 止损移至成本价 |
| 成交量过滤 | 成交量 < 1.2倍均量 | 跳过该信号 |
| ATR过滤 | ATR/价格 < 0.15% | 跳过（波动太小）|
| 低流动性警告 | UTC 22-01点、06-08点 | 日志警告 |

### 用户风控建议

1. **首次实盘**：先用小资金（50-100U）测试
2. **杠杆控制**：新手建议 ≤5x，经验丰富 ≤10x
3. **仓位管理**：单次开仓不超过本金 30%
4. **止损纪律**：永远不要移除止损
5. **定期提现**：盈利后定期转出部分利润

### 不适合交易的情况
- 重大新闻事件前后（非农数据、利率决议等）
- 价格剧烈震荡、无明显趋势时
- 网络连接不稳定时
- 无法持续监控时（虽然机器人自动运行，但建议定期查看）

## 常见问题

### Q1: 为什么有信号但没有开仓？
可能原因：
- 趋势确认未通过（逆势）
- 评分 < 65 分
- 成交量不足（< 1.2倍均量）
- 处于冷却期（刚止损后30分钟）
- 处于低流动性时段

### Q2: 为什么止盈止损价格和设置的不一样？
使用的是**动态 ATR 计算**：
- 止损 = 入场价 ± ATR × ATR倍数
- 止盈 = 入场价 ± ATR × ATR倍数 × 2
- 根据市场波动自动调整，而非固定百分比

### Q3: 模拟模式和实盘有什么区别？
| 功能 | 模拟模式 | 实盘 |
|------|---------|------|
| 资金 | 虚拟资金 | 真实账户资金 |
| 下单 | 仅模拟计算 | 真实提交到交易所 |
| API Key | 可不填 | 必须正确配置 |
| 持仓 | 本地模拟 | 读取交易所真实持仓 |

### Q4: 可以修改已经开仓的止损止盈吗？
可以。在持仓卡片的"手动控制"区域：
1. 输入新的止损或止盈价格
2. 点击确认按钮
3. 新的价格立即生效

### Q5: 机器人会一直运行吗？
会一直运行直到：
- 手动点击"■ 停止机器人"
- 触发风控熔断（亏损超限）
- 软件被关闭

### Q6: 如何导出交易记录？
1. 切换到"📊 交易记录"标签
2. 点击"⬇ 导出 CSV"按钮
3. 文件自动保存到下载文件夹
4. 可用 Excel 打开分析

### Q7: 三种趋势模式有什么区别？
| 模式 | 时间帧 | 特点 | 适用 |
|------|--------|------|------|
| 1h保守 | 1小时 | 信号少但准，过滤震荡 | 稳健型 |
| 30m激进 | 30分钟 | 信号多，捕捉更多机会 | 积极型 |
| 自适应 | 自动切换 | 趋势强用1h，弱用30m | 全能型 |

### Q8: 为什么显示"API 连接异常"？
检查项：
1. 网络是否正常
2. API Key/Secret 是否正确
3. API 权限是否包含"合约交易"
4. Gate.io 服务器是否正常

### Q9: 可以同时开多个方向吗？
不可以。策略设计为单向持仓：
- 有多单时不会开空单
- 需要先平仓才能反向开仓

### Q10: 移动止损和保本止损冲突吗？
不冲突，两者独立运作：
- **保本止损**：一次性触发，盈利达标即移至成本价
- **移动止损**：持续跟踪，随着盈利扩大不断上移止损位
