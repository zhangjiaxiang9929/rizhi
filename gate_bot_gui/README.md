# Gate.io BTC 机器人 GUI 版本

## 快速开始

### 第一步：安装依赖

```
npm install
```

### 第二步：运行（开发模式）

```
npm start
```

### 第三步：打包成 exe

```
npm run build
```

打包完成后在 `dist/` 目录找到安装包 `Gate BTC 机器人 Setup x.x.x.exe`

---

## 界面说明

- 左侧：填写 API Key、调整参数、启动/停止
- 右侧上方：实时价格、余额、信号、倒计时
- 右侧中间：持仓状态（入场价/止损/止盈/浮盈）
- 右侧下方：滚动日志

## 注意

- 打包 exe 约需要 5-10 分钟，文件约 80-150MB
- API Secret 不会保存到磁盘，每次启动需重新输入
- 模拟模式价格固定为 70000（模拟），真实模式拉取实时价格
