# Crypto Learning

Cryptocurrency self-study system. Pushes learning content every morning at 9AM; each invocation produces beginner-friendly detailed reports (not just titles), integrating the latest information via web_search. Includes a complete learning syllabus (beginner, investment, and advanced stages) with progress tracking, skip, and reset support.
